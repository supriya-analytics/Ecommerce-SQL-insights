# Ecommerce Data Analysis Project

## Project Overview
This project contains a detailed analysis of an ecommerce dataset using **4 related tables**.  
The goal is to explore sales patterns, customer behavior, product performance, and category insights to help businesses make data-driven decisions.

## Database Tables

### 1. **Orders**
This table contains all the sales transactions made by customers.

**Columns:**
- Order_ID
- Customer_ID
- Product_ID
- Order_Date
- Quantity
- Sales
- Profit
- Shipping_Mode
- Region
- Segment

- ### 2. **Customers**
This table contains customer details and demographics.

**Columns:**
- Customer_ID
- Customer_Name
- Segment
- Region
- Country

### 3. **Product**
This table contains details about products sold.

**Columns:**
- Product_ID
- Product_Name
- Category_ID
- Sub_Category
- Price

### 4. **Categories**
This table contains category and sub-category information.

**Columns:**
- Category_ID
- Category_Name
- Sub_Category

## Key Insights (Example)
1. **Top selling products** are in the Electronics and Fashion categories.  
2. **Consumer segment** generates the highest sales volume.  
3. **West region** has the highest sales but lower profit due to shipping costs.  
4. **High profit categories** are Technology and Office Supplies.  
5. **Discounts and returns** affect profit even when sales are high.
